### sample-server-druid
此模块在sample-server模块基础上整合Druid

#### druid依赖
```
<dependency>
    <groupId>com.alibaba</groupId>
    <artifactId>druid-spring-boot-starter</artifactId>
    <version>1.1.18</version>
</dependency>
```

#### druid访问方式
```
地址: http://localhost:10084/druid/login.html
账号：root
密码：root
```
